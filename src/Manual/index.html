Introduction

HB5Tool is a program for viewing and manipulating game data for HardBall 5. In its current state, you can't do much with it, but this will change over time.

Due to the early state of the program, this manual is still under construction.

This program is meant to work with multiple data sources. Some of these are user-facing files, like player exports, team exports, and leagues. Other files are meant to be internal, or not edited by the player directly. These files would typically be extracted from HARDBALL.BIN or HB5CDROM.BIN. It is important to keep this "user-facing" vs. "internal" divide in mind when using HB5Tool. Some tasks will require extracting files from an archive, editing them, and then re-inserting them into the same archive.

Notes, Warnings, Precautions, etc.

The primary target for this program is PC versions of HardBall 5. HardBall 4 files are kind of supported, though you may need to rename a few files to get HB5Tool to load them.

As this program targets the PC version, data is assumed to be little endian. This can cause issues when trying to open PlayStation HardBall 5 data.

User-Facing Data

This portion of the manual covers the user-facing data files.

Player Exports

HardBall 5 allows you to export a player to a file. The player type is typically defined by the file extension:

  • .btr – Batters
  • .pit – Pitchers

However, the player export files have a value at the beginning that prevents players from just changing the file extension. (This value also differentiates HardBall 4 player exports from HardBall 5 ones.)

In addition to the "standard" data (e.g. jersey number, name, skin color, etc.) and ratings data, historical stats are also stored in player export files.

Team Exports

Entire teams can be exported from HardBall 5. Team exports have a .hb5 file extension. Players on the roster are also included in the team export data, including their historical stats.

Leagues

Leagues are a combination of players, teams, and other data. League files have a .lgd file extension.

Internal Data

This portion of the manual covers the internal data files. Most of these are not meant to be edited by the player directly. Some files are partially user-facing, as their data can be changed in-game (e.g. the Options and Customization settings).

Main "Glue" Files

A "glue" file is an archive containing multiple files "glued" together. The two primary "glue" files are HB5CDROM.BIN and HARDBALL.BIN. These two files contain the bulk of the game data.

Customization Data

HardBall 5 lets you customize league names and logos, along with various other names. This data lives in CUSTOM.BIN.

Settings/Options

Options are stored in DEFAULTS.BIN.

"Digs" Files

"Digs" is short for "digital sounds".

Program Settings

There are very few program settings, and only one of them actually works right now!

  • Hardball 5 Path – currently unused. Would be used for reading various files in a HardBall 5 install.
  • Default PICS.BIN Path – Path to a PICS.BIN (or similar) file with HardBall 5-format player pictures.
  • Override Default League – currently unused. If "HardBall 5 Path" support were to read DEFAULTS.BIN to get the current active league, this would let you override that league.
